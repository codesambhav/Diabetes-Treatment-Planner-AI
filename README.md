# 🧬 Personalized Diabetes Treatment Planner & Predictive Model

### 🌟 Project Overview
This project is a high-fidelity clinical decision-support tool designed to provide personalized diabetes treatment recommendations. It bridges the gap between raw medical data and clinical action by integrating a **Machine Learning (ML) predictive engine** directly into an interactive **Power BI dashboard**. 

Using a pilot dataset of 50 patients, the system allows healthcare providers to simulate real-time treatment outcomes by adjusting specific patient vitals.

---

### 🚀 Key Technical Features
* **Predictive ML Engine:** Utilizes a **Random Forest Classifier** to recommend treatments (Metformin, Insulin, GLP-1, etc.) based on complex patient profiles.
* **Interactive "What-If" Simulation:** Employs a DAX-based parameter approach (`patientInputTable`) that feeds user inputs into a live Python-driven model for real-time re-calculation.
* **Hybrid Logic Layer:** Custom DAX measures (e.g., `[Best Recommended Treatment]`) translate high-dimensional ML predictions into clear, actionable clinical insights.
* **Professional Version Control:** Developed using the **Power BI Project (.pbip) folder approach**, ensuring modularity, human-readable code, and Git-readiness.

---

### 📊 Model Performance & Validation
The model was rigorously validated using an **80/20 train-test split** to ensure clinical reliability.

* **Overall Accuracy:** **70%** (A robust baseline for a 50-patient clinical pilot).
* **Insulin Recall:** **100%** — The model successfully identified every patient requiring critical insulin intervention.
* **High Precision Targets:** Achieved **100% precision** for Lifestyle and GLP-1 recommendations.
* **Primary Predictors:** The model's decisions are primarily driven by **HbA1c**, **Fasting Glucose**, and **BMI**.
